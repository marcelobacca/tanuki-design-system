DESIGN SYSTEM

Tanuki Sensei

Design system do app nativo Tanuki Sensei — um companion leve e cozy para iniciantes em japonês. Feito em React Native (iOS + Android), então é mobile-first. As foundations e componentes são reaproveitáveis caso surja uma superfície web no futuro — não são exclusivos de mobile, mas os padrões e telas documentados aqui assumem contexto de app.
