# Ícones — Untitled UI Icons

**Biblioteca:** Untitled UI Icons · 1.175 ícones · stroke-based
**Componente:** `src/components/Icon.js`
**Uso:** `<Icon name="nome-do-icone" size={20} color={T.text} />`

## Tamanhos padrão
| Contexto | Size |
|---|---|
| Nav bar | 20px |
| Botões / inputs | 18–20px |
| Top bar sessão | 16px |
| Badges / micro | 14px |

## Regra: cor por estado
- Ativo/primário: `T.pri`
- Default: `T.text`
- Secundário/inativo: `T.muted`
- Desabilitado: `T.dim`
- Erro: `T.err`
