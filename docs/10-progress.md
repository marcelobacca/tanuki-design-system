# Progress Bars

- Track: bg `T.track` (#2E2C3F) · radius R.pill
- Fill: radius R.pill · transition width 0.6s

## Tamanhos
| Size | Height |
|---|---|
| sm | 6px |
| md | 8px (padrão — Session header) |
| lg | 12px |

## Cores semânticas da fill
- Normal/progresso: `T.pri`
- Aviso: `T.warn`
- Completo/100%: `T.ok`
