# Cores — Design Tokens

Definidos em `src/theme.js` como `T.*`. Dark mode only.

## Superfícies
| Token | Hex | Uso |
|---|---|---|
| `bgDeep` | #06050E | Auth, hero, onboarding |
| `bg` | #0F0E18 | Background base de todas as telas |
| `surf` | #1A1929 | 1ª elevação — cards, header, inputs, sheet |
| `surf2` | #211F32 | 2ª elevação — menus, popovers |
| `surfElevated` | #271E3C | Session-complete, missões, streak modal e cards que se movem juntos |
| `bgMissions` | #0F062B | Exclusivo MissionsScreen |

## Texto
| Token | Hex | Uso |
|---|---|---|
| `text` | #E8E6F8 | Principal |
| `textSec` | #A8A6C8 | Secundário |
| `muted` | #7A78A3 | Placeholders, labels de apoio |
| `dim` | #4E4C6A | Inativos, hints sutis |
