Estrutura de Zonas

A SessionScreen é dividida em 5 zonas determinadas. Cada zona tem altura própria ou flex, com 12px de espaçamento entre elas. As zonas 3 e 4 garantem posição vertical consistente dos botões de resposta em todos os tipos de exercício.

Zona 1 — Barra superior

Altura determinada pelo conteúdo. Gear 44×44 + progress bar flex:1 + timer. paddingVertical:10 · paddingHorizontal:16.

Zona 2 — Instrução ↑ 12px ↓

Texto da instrução. 12px uppercase Bold muted. paddingVertical:8. marginBottom:12 separa da zona 3.

Zona 3 — Prompt ↑ 12px ↓

Prompt + legenda como grupo único centralizado. Legenda sempre renderizada (opacity:0 se ausente) para não deslocar zona 4. marginBottom:12.

Tipo 12: minHeight:24px (espaçador) · Tipo 14: inclui prompt romaji

Zona 4 — Respostas

flex:1 — ocupa todo espaço restante. paddingHorizontal:16 · paddingTop:12 · alignItems:flex-start. Sempre começa no mesmo Y.

MC grid · input · build-word tiles · audio button · canvas (tipo 14)

Zona 5 — Rodapé

Fora do ScrollView. padding:16 paddingBottom:32. Feedback de acerto/erro + botão de ação (Próximo / OK / Ver resultados).
